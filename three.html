算術演算子

　+(足す)、-(引く)、*(かける)、/(割る)、%(余り)のこと。


#include <iostream>
#include <string>


int main(void) {
	const int a = 5;
	const int b = 2;

	std::cout << a+b << " " << a-b << " " << a*b << " " << a/b << " " << a%b << std::endl;

	std::string c = "sample";
	std::string d = "code";

	std::cout << c+d << std::endl;

	int e = a+b;

	e++;
	
	std::cout << e << std::endl;

	e+=10;
	std::cout << e << std::endl;

	e--;
	std::cout << e << std::endl;

	return 0;
}

サンプル解説

a+b,a-b,a*b
そのまま。思ったまんまの答えが出ると思うよ。
a/b
いわゆる割り算の商を出す。整数同士の場合、少数は切り捨てられる。
a%b
いわゆる割り算の余りを出す。整数同士でしか使うことができない。
e++,e--
eの値を1増やす。1減らす。
e+=10
eの値を10増やす。-,*,/,%版もあるがあんまり使わないため詳しいことは自分で確かめてみよう。

c+d
文字列同士を組み合わせ、別の文字列を作ることができる。+しかないので注意。
