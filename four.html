目標

　コンソール画面から文字、数字を入力できるようになる。


語句説明

標準入力

　コンソール画面でのキーボードによる入力のこと。

サンプル

#include <iostream>
#include <string>

int main(void){
  int a,b;

  std::cin >> a >> b;
  std::cout << a+b << " " << a-b << " " << a*b << " " << a/b << std::endl;


  std::string c,d;
  std::cin >> c >> d;
  std::cout << c+d << std::endl;

  return 0;
}

サンプル解説

std::cin

　std::cinは>>の後にある変数にコンソール画面に入力された値を入れることができます。
　入力するもの同士は>>で区切ります。入力する際はコンソール画面に値を入れた後Enterキーを押してください。


std::cin >> a >> b;
a,bにそれぞれ値が入力されます。改行やスペース、タブが入力の区切りとして扱われます。


課題
  • ・サンプルコードを実行して出力を確かめる。
  • ・名前を入力すると「○○さん、こんにちは。」と表示されるようにする。
  • 要提出・5教科の点数をそれぞれ入力すると平均を出力するプログラムを作る。
